3. Lua egitiga dengan ala 18cm mempunyai lua 135cm². Tinggi egitiga terebut adalah.


Pembahasan

Diketahui:

a = 18 cm

L = 135 cm²

Ditanya:

t = ?

Jawaban:

[tex] \sf L = \frac{a \times t}{2} [/tex]

[tex] \sf 135 = \frac{18 \times t}{2} [/tex]

[tex] \sf 135 \times 2 = 18t[/tex]

[tex] \sf 270 = 18t[/tex]

[tex] \sf t = \frac{270}{18} [/tex]

[tex] \sf t = 15 \: cm[/tex]

Kesimpulan:

Jadi, tinggi segitiga tersebut adalah 15 cm.

35. Menghitung Lua Segi Tiga Lua = 1/2 ala x tinggi algoritma.


Berikut ini adalah algoritma untuk menghitung luas segitiga:

   Baca panjang alas segitiga (alas) dan tinggi segitiga (tinggi)    Hitung luas segitiga dengan rumus: luas = (1/2) x alas x tinggi    Tampilkan hasil perhitungan luas segitiga

Kamu dapat menggunakan algoritma tersebut untuk menghitung luas segitiga dengan mudah. Setelah kamu memahami algoritma tersebut, Anda dapat menuliskannya dalam bentuk kode program yang dapat dijalankan di komputer. Jika kamu menggunakan bahasa pemrograman python, maka kode programnya dapat ditulis sebagai berikut:

# Baca panjang alas segitiga (alas) dan tinggi segitiga (tinggi)

alas = float(input("Masukkan panjang alas segitiga: "))

tinggi = float(input("Masukkan tinggi segitiga: "))

# Hitung luas segitiga dengan rumus: luas = (1/2) x alas x tinggi

luas = (1/2) * alas * tinggi

# Tampilkan hasil perhitungan luas segitiga

print("Luas segitiga adalah:", luas)

Contoh lain dalam rust:

use std::io;

fn main() {

   // Baca panjang alas segitiga (alas) dan tinggi segitiga (tinggi)

   println!("Masukkan panjang alas segitiga: ");

   let mut alas = String::new();

   io::stdin().read_line(&mut alas).expect("Gagal membaca input");

   let alas: f64 = alas.trim().parse().expect("Input bukan angka");

   println!("Masukkan tinggi segitiga: ");

   let mut tinggi = String::new();

   io::stdin().read_line(&mut tinggi).expect("Gagal membaca input");

   let tinggi: f64 = tinggi.trim().parse().expect("Input bukan angka");

   // Hitung luas segitiga dengan rumus: luas = (1/2) x alas x tinggi

   let luas = (1/2) * alas * tinggi;

   // Tampilkan hasil perhitungan luas segitiga

   println!("Luas segitiga adalah: {}", luas);

}
