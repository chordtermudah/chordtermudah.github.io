8. Besar ∠ PRS pada gambar di bawah ini adalah .... ​


1 segitiga = 180°

∠PRS = 180° - ( 45° × 2 )

∠PRS = 180° - 90°

∠PRS = 90°

Konsep:
"Total besar sudut dalam segitiga adalah 180°"
- - - - -
Maka:
<RPS + <PSR < PRS = 180°
45° + 45° + <PRS = 180°
90° + <PRS = 180°
<PRS = 180° - 90°
<PRS = 90°

Jadi, besar <PRS adalah 90°

39. ditanyanya luas segitiga PRS


Penjelasan dengan langkah-langkah:

dik: s= 17 cm

a?

a= √17^(2)-8^(2)

=√289-64

=√255

=15

alas-5=10

tanya: L segitiga?

jwb:

L Segitiga =1/2×a×t

=1/2×10×8

=40 cm^(2)

Jawab:

40 cm²

Penjelasan dengan langkah-langkah:

Luas segitiga PRS = Luas segitiga PRQ - Luas segitiga SRQ

L = 1/2.PQ.QR - 1/2.SQ.QR

L = 1/2(PS+SQ)QR-1/2(SQ)QR

L = 1/2.QR[PS+SQ-SQ]

L = 1/2.QR.PS

mencari nilai PS menggunakan pyhtagoras

PQ² = PR²-QR²

(PS+5)² = 17²-8²

(PS+5)² = 225

PS+5 = ±√225

PS+5 = ±15          ...pilih yang positif

PS+5 = 15

PS = 10 cm

maka

Luas segitiga PRS = 1/2.QR.PS

                              = 1/2.8.10

                              = 40 cm²
