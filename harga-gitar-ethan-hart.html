7. Seperempat bagian dari uang Ethan digunakan untuk membeli makan. 23 dari sisanya disumbangkan untuk korban banjir. Uang Ethan sekarang tersisa Rp130.000,00. Berapa jumlah uang Ethan mula-mula?​


Jawaban:

Uang Ethan mula-mula = Rp 520.000,00

Penjelasan dengan langkah-langkah:

• Uang tersisa = Rp 130.000,00

• Uang yang disumbangkan ⅔

Uang yang tersisa berarti ⅓ bagian

Sehingga banyaknya uang yang disumbangkan ⅔ bagian

= Rp 130.000,00 × 2

= Rp 260.000,00

• Jumlah uang sebelum disumbangkan

= Uang yang disumbangkan + uang tersisa

= Rp 260.000,00 + Rp 130.000,00

= Rp 390.000,00

• Jumlah uang untuk makan

Uang sebelum disumbangkan ¾ bagian

Sehingga banyaknya uang untuk makan ada ¼ bagian dari uang yang disumbangkan

= Rp 390.000,00 / 3

= Rp 130.000,00

• Jumlah uang mula-mula

= Uang untuk makan + uang sebelum disumbangkan

= Rp 130.000,00 + Rp 390.000,00

= Rp 520.000,00

27. Ethan berjalan ke arah timur sejauh 4 meter dan berkelok ke arah utara sejauh 3meter. tentukan jarak dan perpindahan ethan


Jawab:

Jarak = 4m + 3m = 7 meter

Perpindahan: jika digambar maka lintasan ethan berjalan membentuk siku siku. sehingga perpindahannya dapat dihitung menggunakan rumus pythagoras.

misal : a sebagai jarak Ethan berjalan ke arah timur

           b sebagai jarak Ethan berjalan ke arah utara

           c sebagai perpindahan

maka,

c = [tex]\sqrt{a^{2} + b^{2} }[/tex]

c = [tex]\sqrt{4^{2} + 3^{2} }[/tex]

c = [tex]\sqrt{16 + 9 }[/tex]

c = [tex]\sqrt{25}[/tex]

c = 5

Jadi, perpindahan Ethan = 5 meter
